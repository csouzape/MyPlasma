# MyPlasma

Personal KDE Plasma Configuration

The goal of this project is to reproduce my complete desktop environment on a fresh Linux installation with minimal manual intervention.

This repository contains:

- Plasma configuration
- KWin configuration
- KDE application settings
- GTK integration
- Themes
- Icons
- Fonts
- Wallpapers
- Package lists
- Installation scripts

## Supported

- Arch Linux


## Installation

```bash
git clone https://github.com/csouzape/MyPlasma
cd MyPlasma
./install.sh
```
